A deque is a queue that let we dequeue it from both ends, normal queue use LIFO, a monotonic one is a deque with a sorted order of the element, in ascending or descending way

We use it for when we need to keep track of a min or a max in a range(for e.g a sliding window), or need to scan a sequence in order and also need to dequeue it from both ends
