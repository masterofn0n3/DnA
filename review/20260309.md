### [739 Daily Temperature](../new/20260309/739_daily_temperatures.md)

- naive approach by bruce force, nested loop for each index: scan one all of its next element to find the next warmer day and increase the counter
- optimize approach: a stack that contain index of current element
  - for each iteration, create a for loop that compare the current element to the top of the stack, if it's greater, set that index to the current element index - the stack top, then pop it from the stack
  - append the current index to the stack
  - in the end, the poped value will have its index updated, and the remaining value in the stack is the day that didn't get any warmer day
